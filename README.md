## Description

- 장르: 퍼즐
- 테트리스와 슈퍼 마리오를 결합한 퍼즐 게임
- 5개의 스테이지 제작

## How To Play
- 테트리스 단계
	- 테트리스 블럭을 쌓거나 지워서 마리오가 이동할 수 있도록 길을 만듦
- 마리오 단계
	- 마리오를 조작하여 골 지점으로 이동

## Development Environment
- Visual Studio
- Windows Forms
- C#
